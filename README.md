# HEPi

Python interface for gluing together several HEP programs (e.g. from HEP-Forge).
Currently slurm is supported for clusters.

## Versions

### Stable

```sh
pip install hepi [--user] [--upgrade]
```

### Dev

```sh
pip install --index-url https://test.pypi.org/simple/ hepi
```
